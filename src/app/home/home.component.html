Stock Market Prediction - ML

Welcome to the Stock Market Prediction GUI

This is a webapp that has been built using MEAN stack, meaning it uses MongoDB for the database, Express Node Server, Angular 4 for the front end and NodeJS for the back end.

I am using an API called quandl that gives recent stock dataset that we can work on. An interactive chart is shown for users to compare between companies and play around with the dataset. Regression techniques are used with the help of a library called ml-regression to predict the future of a company.

Note: This app predicts only for future years. Short term prediction like months will be for future enhancement.
