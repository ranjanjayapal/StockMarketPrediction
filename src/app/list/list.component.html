Stock Markets are very volatile. Predicting stocks a few year(s) ahead is a task that can and has been achieved.
But predicting a few months ahead is too short a time, and is a challenging task even data scientists face.
Predicting using existing ML models would give data that is very unreliable.
Hence this feature will be released in a future update once I work on different machine learning and deep learning models to predict in months.
